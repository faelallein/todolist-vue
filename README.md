# todolist-vue

## 1 - Rodar a Api

```
Abrir a pasta api pelo cmd e executar java -jar api-todolist.jar

API utiliza H2 como dependência, mantem as informações enquanto esta rodando.
```

## 2 - Project setup

```
npm install
```

## 3 - Compiles and hot-reloads for development

```
npm run serve
```
